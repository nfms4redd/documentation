Arquitectura
============

La arquitectura del cliente web viene definida en el cliente principalmente por el uso del proyecto RequireJS y por el uso del patrón message-bus. La parte servidora se basa en la especificación 3 de Java Servlets. A continuación se presentan los conceptos básicos de estas tecnologías para terminar presentando el caso concreto del portal REDD.
